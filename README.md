## Overview

This is a third party tool for FinalFantasy14.
Outputs the specified user and their role at random.
Suitable for romping with friends!

## System Requirements

| System | Version |
| ------ | ------- |
| TypeScript | v5.1.6 |
| Nextjs | v13.4.19 |
| TailWindCSS | v3.3.3 |

## Run Server

### For Dev
```bash
cd src

yarn dev
```
